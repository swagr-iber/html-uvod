Co to je HTML

HTML je odkazovací značkovací jazyk pro tvorbu webových stránek. HTML značky (tagy) v html dokumentu vyznačují, která část textu je nadpisem, která normálním odstavcem nebo hypertextovým odkazem, vyznačují místo, kam má být umístěn obrázek či video, apod.

Značky jsou obvykle párové, přičemž koncová značka je shodná se značkou počáteční, jen má před názvem znak lomítko (např. html, head, body, h1, h2, p, a, ul, ol, div). Některé značky jsou nepárové - nemají žádný obsah a nepoužívají koncovou značku (např. hr, br, img, meta, link).

Klíčové vlastnosti jazyka

  • Webový prohlížeč ignoruje nadbytečné mezery v textu (zůstane jen jedna)
  • Enterem nejde odřádkovat (k tomu je CSS)
  • Webový prohlížeč ignoruje vše co nezná (přeskočí to)
  • Veškerý text musí být umístěn v nějaké značce

Některé značky obsahují tzv. atributy a jejich hodnoty. 

    <a href="cil odkazu">co je klikatelné</a>
    <img src="obrazek.jpg">

Co to je CSS

CSS jsou kaskádové styly, pomocí nichž určujeme vzhled webových stránek. Definujeme styly pro nadpisy, odstavce, odkazy či obrázky, ale i pro jednotlivé bloky webové stránky.

Definice kaskádových stylů sestává z několika pravidel. Každé pravidlo obsahuje selektor a blok deklarací uzavřený ve složených závorkách {}. Každý blok deklarací pak obsahuje deklarace oddělené ; (středníky) a každá deklarace sestává z vlastnosti, následuje : (dvojtečka) a její hodnoty. 

    selektor {
      vlastnost: hodnota;
      vlastnost: hodnota;
    }

Propojení HTML a CSS

Web je složen z řady html stránek, které mají stejný vzhled. Tento vzhled je vytvořen pomocí stylů zapsaných v jednom společném CSS souboru. Ve všech HTML stránkách je pak uvedeno propojení (link) na tento CSS soubor.

Díky tomu mají stránky jednotný vzhled a každá změna stylu v css souboru se projeví ve všech html stránkách, které jsou s tímto css souborem propojeny.

Shrnutí

  • Pomocí HTML definujeme význam elementů: Nadpis, odstavec, odkaz, obrázek… a samozřejmě jejich pořadí na stránce (jako v klasickém dokumentu).
  • Pomocí CSS definujeme vzhled elementů: Nadpis bude zarovnaný na střed, odstavec bude červeným písmem, odkaz nebude podtržený, obrázek bude odsazený shora o 20px…

Neplatí absolutně a vždy. Jako všude i zde najdete sem tam nějaké ty výjimky.
